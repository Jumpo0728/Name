# Installation Guide (Bedrock Edition)

This folder (`minecraft_resource_pack/`) is a standard Minecraft Bedrock **resource pack**.

## Option A: Import as a .mcpack

1. Zip the contents of `minecraft_resource_pack/`.
2. Rename the archive extension from `.zip` to `.mcpack`.
3. Open the `.mcpack` file on your device to import it into Minecraft.
4. In Minecraft: **Settings → Resource Packs**, activate the pack.

## Option B: Manual install

1. Locate your `resource_packs` directory.
2. Copy the entire `minecraft_resource_pack` folder into `resource_packs/`.
3. In Minecraft: **Settings → Resource Packs**, activate the pack.

## Notes

- If you update the pack files, restart Minecraft to ensure everything reloads.
